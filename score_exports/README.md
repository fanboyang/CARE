# Score Export Directory

This release expects fixed confidence-score exports generated by the scoring
models used for the reported experiments. Large score files are not included.

Use this directory structure:

```text
score_exports/
└── <score_model>/
    └── <dataset>/
        └── seed<id>/
            ├── calibration_val.csv
            ├── calibration_neg.csv
            ├── test_pos.csv
            ├── test_neg.csv
            └── test_with_neg.csv
```

Supported score models:

```text
ukge, passleaf, beurre
```

Supported datasets:

```text
cn15k, nl27k, ppi5k
```

Each CSV must contain these columns:

```text
h,r,t,label,score,source_split
```

`score` is the fixed model confidence score. `label` is the gold confidence
value used for conformal calibration and evaluation.
